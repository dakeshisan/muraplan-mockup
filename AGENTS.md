## ATLAS — контекст проекта

Единый ERP-портал застройщика ATAMŪRA GROUP (Казахстан). Стек: TanStack Start v1
+ React 19 SSR, Tailwind v4 (oklch, палитра Navy+Teal в `src/styles.css`), shadcn
(new-york), bun. Все данные — демо/mock; логин фейковый (любой пароль).

**Читай `CODEX_START.md`** — там полная карта: как запустить, архитектура портала
(3 пространства, 9 ролей, ось объекта, ⌘K), список 19 экранов и следующие шаги.

Перед «готово» — `bun run build` должен быть зелёным.
